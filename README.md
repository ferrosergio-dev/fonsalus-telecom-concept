## Overview
A modern, responsive website concept for a maritime telecommunications company. This design demonstrates how an outdated maritime telecom website can be transformed into a contemporary, user-friendly platform.

## Features
- **Modern Design**: Clean, professional maritime-themed interface
- **Full Responsiveness**: Works on all devices (desktop, tablet, mobile)
- **Complete Navigation**: All 7 main service areas from the original site
- **Interactive Elements**: Slider, modals, forms with validation
- **Easy Customization**: CSS variables and config files for quick rebranding
- **Client Portal Concept**: Modern approach to client accounts

## Customization for Your Company

### Quick Branding Changes:
1. **Company Name**: Replace "Fonsalus" with your company name in:
   - `index.html` (search for "Fonsalus")
   - `script.js` (COMPANY_CONFIG object)
   - CSS variables in `style.css`

2. **Colors**: Modify CSS variables in `:root` section of `style.css`:
   ```css
   :root {
       --color-primary: #1a5f9c;      /* Your primary color */
       --company-name: "Your Company";
   }
   ```
3. **Logo:** Replace placeholder logo with your actual logo files
